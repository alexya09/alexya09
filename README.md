# 💫 About Me:
Sou Estagiária de Curadoria e Dados, apaixonada por otimizar a experiência do usuário através da Inteligência Artificial. Meu trabalho envolve desde o design conversacional e treinamento de NLU para aprimorar chatbots, até a extração de dados (SQL) para a construção de dashboards e acompanhamento de KPIs estratégicos no Power BI. Adoro transitar entre a análise de dados e o desenvolvimento de soluções tecnológicas.
